**Data Types**
```python
# int: Whole Numbers
this_is_a_int = 5

# float: Decimals Numbers
this_is_a_float = 3.14

# string: Words, Sentences, Letters
this_is_a_string = "Hello Reader!"

# bool: True or False
this_is_a_boolean = True

# list: Arrays 
this_is_a_list = ["Python", "C++", "R", 12, True, 3.14]

# dict: Similar to Arrays but use Key-Value
this_is_a_dictionary = {1: "Wikipedia", 2: "YouTube", 3:"W3Schools"}

```


**Math Operators**
```python
# + for Addition
print(3 + 5) #8

# - for Subtraction
print(5 - 5) #0

# * for Multiplication
print(2 * 2) #4
 
# / for Division
print(4 / 2) #2

# // for Integer Division
print(25 // 2 ) #12

# % for Modulo
print(25 % 2) #1
 
# ** for Exponent
print(3 ** 3) #27
```

**Print**
```python
# We can "Print" or push things onto the Terminal. We can print out datatypes or 
# variables. Whatever we want to "See" the outcome of in the terminal, we Print

# Printing Data Types
print(5)
print(3.14)
print("Hello World")
print(True)

# Printing Variables
var_one = 7
var_two = 3
var_three = "Hello World"

print(var_one)
print(var_two)
print(var_three)

# We can even use our Math Operators, just like you've seen earlier
print(var_one + var_two)
print(2+3)

```
